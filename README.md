## How to run
If you would like to modify the source code or build from source clone this repository via:
```shell
git clone https://github.com/Kamiya1337/Tsukasa-client-showcase.git
```
The resulting folder is a gradle project that can opened with your favorite IDE i.e. IntelliJ. 

To run the client from inside IntelliJ run the "run" gradle task
