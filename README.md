## 🛠️ What We Did & Learned

In this project, my co-developer and I used the standard MCP environment to decompile the application, understand its codebase, and inject our own custom modules.

### Key Work & Skills:
* **Feature & Exploit Development:** Developed and integrated custom Java modules to override default system behaviors and inject new functionalities into the runtime application.
* **Code Auditing:** Analyzed decompiled Java classes to find correct hooks and event triggers for our custom kịch bản (scripts).
* **Collaboration with Git:** Managed the project development with a teammate via Git, practicing how to branch, commit, and resolve code conflicts properly.

## 🧰 Tech Stack
* **Language:** Java (97.8%)
* **Tools Used:** MCP (Minecraft Coder Pack) for environment setup, Git/GitHub for version control.
